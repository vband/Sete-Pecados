DESCRIPTION
-=========-
Original asymmetrical pixel font. Size of base symbol is 7*12 pixels. Latin and Cyrillic code pages are supported.

Files in double_pixel-7.zip:
       	readme.txt     				this file;
        double_pixel-7.ttf    			regular font;
	double_pixel-7_windows_1251.fon   	font in windows "FON" format Cyrillic codepage;
	double_pixel-7_windows_1252.fon   	font in windows "FON" format Latin codepage;
	double_pixel-7_screen.png		preview image.
